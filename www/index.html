sampleSelection

Tools for estimating sample selection models in R

What is sampleSelection?

  • sampleSelection is an extension package for the "language and environment for statistical computing and graphics" called R.
  • sampleSelection provides functions for estimating sample selection models (also known as generalized tobit or Heckman selection models).

What can sampleSelection do?

If a non-random (sub)sample is used for econometric estimation, Heckman selection models are required to obtain unbiased estimates. sampleSelection provides functions for:
  • two-step Heckman estimation
  • Maximum Likelihood estimation (using package maxLik)
  • variance-covariance matrices for Heckman-type estimations
  • "Inverse Mill's Ratios" for standard selection models (from univariate probit estimations)
  • "Inverse Mill's Ratios" for double-selection models (from bivariate probit estimations)

Where can I get sampleSelection?

  • The released version is available on CRAN
    • You can download the source code, binaries for MacOS X and MS-Windows, the reference manual, and a vignette from http://cran.r-project.org/web/packages/sampleSelection/index.html.
    • You can install the package by typing at the R prompt (if you have internet access): 
       install.packages( "sampleSelection" )
  • The current development version is available on R-Forge
    • You can download the source code and binaries for MacOS X and MS-Windows from http://r-forge.r-project.org/R/?group_id=256.
    • You can install the package by typing at the R prompt (if you have internet access): 
       install.packages( "sampleSelection", repos="http://R-Forge.R-project.org" )
